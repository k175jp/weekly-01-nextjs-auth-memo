# weekly-01-nextjs-auth-memo

Simple authentication-based memo application built with Next.js.

This project was created as part of my weekly development series.

## Features
 - Simple login flow
 - Protected pages
 - Create / read / delete memos
 - Local storage persistence
 - Responsive UI

## Tech Stack
 - Next.js
 - React
 - TypeScript
 - Tailwind CSS
 - Docker
 - Docker Compose

## Getting Started
### Run locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

### Run with Docker
```bash
docker compose up
```
Open http://localhost:3000

## Purpose
The goal of this project was to practice:
 - Next.js App Router
 - Authentication flow
 - Route protection
 - State management
 - Component-based UI development
 - Docker-based development environment

## Notes
This is a small experimental project focused on learning and implementation rather than production readiness.
